Doctor's Portfolio Website

Built a full-stack doctor portfolio and booking platform with dynamic content, appointment booking, and a secure admin panel. Deployed on Vercel and Render with MongoDB Atlas and cloud media integration.

Live Link

Project Overview

This project is a full-stack doctor portfolio and booking platform built with Next.js (React, TypeScript) on the frontend and Node.js (Express.js) for the backend REST API. It aims to provide doctors with a professional online presence while allowing patients to book appointments easily.

The platform features dynamic content rendering for sections like services, about, and gallery. It includes a functional appointment booking system and a contact form, all backed by MongoDB with Mongoose for structured data storage.

A secure admin panel was developed using JWT-based authentication, enabling administrators to manage services, content, and gallery images with Cloudinary integration for media uploads. Admins can also view bookings and user messages.

The frontend was deployed on Vercel and the backend on Render, with both connected to a MongoDB Atlas cloud database. The project also involved managing environment variables for seamless deployment across different stages.
